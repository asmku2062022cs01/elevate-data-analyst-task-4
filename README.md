# Project Overview

This project focuses on designing an interactive business dashboard using Power BI to help stakeholders understand sales and profit performance. The dashboard highlights KPIs, trends, regional performance, and product-based revenue to support data-driven decision-making.

# Objective

To build a dashboard that enables business leaders to:

Monitor Sales, Orders, and Growth %

Understand monthly sales trends

Identify top-performing and underperforming product lines

Analyze country/region-wise performance

Take informed business decisions based on insights
